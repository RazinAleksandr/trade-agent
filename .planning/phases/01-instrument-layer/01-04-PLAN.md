    **lib/trading.py** (per INST-05, INST-06, D-09, D-10, D-11, SAFE-02, SAFE-05):

    Import: `uuid`, `ClobClient` from `py_clob_client.client`, `OrderArgs` and `OrderType` from `py_clob_client.clob_types`, `BUY` from `py_clob_client.order_builder.constants`, `OrderResult` from `lib.models`, `DataStore` from `lib.db`, `get_fill_price` from `lib.pricing`, `get_logger` from `lib.logging_setup`

    Use stateless functions (not a class) so CLI tools can call them directly:

    - `def validate_order(price: float, size: float, order_min_size: float = 5.0) -> tuple[bool, str]`:
      - Check `price > 0 and price < 1`: if not, return `(False, "Price must be between 0 and 1")`
      - Round size to 2 decimal places: `size = round(size, 2)` (SAFE-05)
      - Check `price * size >= order_min_size`: if not, return `(False, f"Order notional {price*size:.2f} below minimum {order_min_size} USDC")`
      - Return `(True, "")`

    - `def execute_paper_trade(market_id: str, question: str, side: str, token_id: str, size: float, host: str, store: DataStore, condition_id: str = "", estimated_prob: float = 0, edge: float = 0, reasoning: str = "", neg_risk: bool = False, order_min_size: float = 5.0) -> OrderResult`:
      - Map side to trade_side: if `side == "YES"`, trade_side = "BUY"; if `side == "NO"`, this is buying NO tokens, so trade_side = "BUY" as well (we always BUY the token we believe in)
      - Get fill price from CLOB API: `fill_price = get_fill_price(token_id, "BUY", host)` (D-09: buys fill at best ask)
        - NOTE: get_fill_price already handles the inversion (BUY trade -> calls get_price SELL -> best ask)
      - If get_fill_price raises (CLOB unreachable): let it propagate -- D-10 says no fake fills
      - Round size to 2 decimals: `size = round(size, 2)` (SAFE-05)
      - Validate: `valid, msg = validate_order(fill_price, size, order_min_size)`. If not valid, return `OrderResult(order_id="", success=False, message=msg, is_paper=True)`
      - Generate order_id: `f"paper-{uuid.uuid4().hex[:12]}"`
      - Record trade: `store.record_trade(market_id=market_id, question=question, side=side, price=fill_price, size=size, token_id=token_id, condition_id=condition_id, order_id=order_id, is_paper=True, estimated_prob=estimated_prob, edge=edge, reasoning=reasoning, neg_risk=neg_risk, fill_price=fill_price)`
      - Update position: `store.upsert_position(market_id=market_id, question=question, side=side, price=fill_price, size=size, token_id=token_id)`
      - Return `OrderResult(order_id=order_id, success=True, message="Paper trade executed", is_paper=True)`

    - `def execute_live_trade(market_id: str, question: str, side: str, token_id: str, price: float, size: float, host: str, private_key: str, chain_id: int, store: DataStore, condition_id: str = "", estimated_prob: float = 0, edge: float = 0, reasoning: str = "", neg_risk: bool = False, order_min_size: float = 5.0) -> OrderResult`:
      - Round size to 2 decimals: `size = round(size, 2)` (SAFE-05)
      - Validate: `valid, msg = validate_order(price, size, order_min_size)`. If not valid, return failed OrderResult.
      - Create authenticated ClobClient:
        ```python
        client = ClobClient(host, key=private_key, chain_id=chain_id, signature_type=0)
        creds = client.create_or_derive_api_creds()
        client.set_api_creds(creds)
        ```
      - Create order: `order_args = OrderArgs(token_id=token_id, price=price, size=size, side=BUY)`
        - We always BUY the token (YES or NO) we believe in, per v1 pattern
      - Sign and post: `signed = client.create_order(order_args)` then `result = client.post_order(signed, OrderType.GTC)`
      - Extract order_id: `result.get("orderID", result.get("id", "unknown"))`
      - On success: `store.record_trade(...)` and `store.upsert_position(...)` with is_paper=False
      - On exception: log error, return `OrderResult("", False, str(e), False)`
