    **lib/portfolio.py** (per INST-07, INST-08, cherry-pick from v1 portfolio.py):

    Use stateless functions (not a class) for CLI compatibility:

    - `def get_portfolio_summary(store: DataStore, gamma_api_url: str, max_total_exposure_usdc: float = 200.0) -> dict`:
      - Get `positions = store.get_open_positions()`
      - Get `total_exposure = store.get_total_exposure()`
      - Get `stats = store.get_strategy_stats()`
      - For each open position, fetch current market price from Gamma API:
        - `market = fetch_market_by_id(gamma_api_url, pos["market_id"])`
        - If market found: set `current_price` based on side (YES -> market.yes_price, NO -> market.no_price)
        - Calculate `unrealized_pnl = (current_price - pos["avg_price"]) * pos["size"]`
        - Update position dict with `current_price` and `unrealized_pnl`
        - Also update in DB: `store.conn.execute("UPDATE positions SET current_price=?, unrealized_pnl=? WHERE id=?", ...)`
      - `store.conn.commit()`
      - Return dict: `{"open_positions": len(positions), "total_exposure_usdc": total_exposure, "remaining_capacity": max_total_exposure_usdc - total_exposure, "positions": positions, **stats}`

    - `def check_resolved_markets(store: DataStore, gamma_api_url: str) -> list[dict]`:
      - Get `positions = store.get_open_positions()`
      - `resolved = []`
      - For each position:
        - `market = fetch_market_by_id(gamma_api_url, pos["market_id"])`
        - If market and `market.closed`:
          - Determine exit_price: YES side -> market.yes_price, NO side -> market.no_price
          - `store.close_position(pos["market_id"], exit_price)`
          - Append to resolved: `{"market_id": pos["market_id"], "question": pos.get("question", ""), "side": pos["side"], "exit_price": exit_price, "avg_price": pos["avg_price"], "size": pos["size"], "realized_pnl": round((exit_price - pos["avg_price"]) * pos["size"], 2)}`
      - Return resolved list

    - `def check_risk_limits(store: DataStore, max_total_exposure_usdc: float = 200.0, max_position_size_usdc: float = 50.0) -> dict`:
      - Get `total_exposure = store.get_total_exposure()`
      - Get `positions = store.get_open_positions()`
      - `warnings = []`
      - If `total_exposure > max_total_exposure_usdc * 0.9`: add warning
      - For each position: if `pos["cost_basis"] > max_position_size_usdc * 0.9`: add warning
      - Return: `{"total_exposure": total_exposure, "max_exposure": max_total_exposure_usdc, "utilization": total_exposure / max_total_exposure_usdc if max_total_exposure_usdc > 0 else 0, "num_positions": len(positions), "warnings": warnings}`

    **tools/get_portfolio.py** (per INST-07, D-01 through D-04):
    - Shebang: `#!/usr/bin/env python3`
    - Docstring: `"""Get current portfolio with open positions and P&L."""`
    - argparse arguments:
      - `--pretty` action="store_true"
      - `--include-risk` action="store_true", help="Include risk limit checks in output" (dest="include_risk")
    - `config = load_config(args)`
    - `register_shutdown_handler()`
    - `store = DataStore(db_path=config.db_path)`
    - Try/finally to close store:
      - Call `get_portfolio_summary(store=store, gamma_api_url=config.gamma_api_url, max_total_exposure_usdc=config.max_total_exposure_usdc)`
      - If `args.include_risk`: also call `check_risk_limits(store, config.max_total_exposure_usdc, config.max_position_size_usdc)` and merge into output
      - Output JSON to stdout
    - On exception: `error_exit(str(e), "PORTFOLIO_FAILED", EXIT_API_ERROR)`
    - `if __name__ == "__main__": main()`
